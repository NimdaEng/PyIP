1. เปิด ไฟล์ config.ini เปลี่ยน ip = <เป็น IP Gateway ของท่าน หรือ ip อะก็ได้ที่สามารถ ping ตลอดเวลาในวง networkของท่าน>
2. ก๊อบปี้โฟลเดอร์ PyIP วางไว้ที่ไดร์  C:\
3. สร้าง shortcut คลิกขวาที่ IP.exe -> send to -> Desktop
4. ดับเบื้้ลคลิกไฟล์ ip-startup.bat ก๊อบปี้ shortcut ที่หน้้า desktop ไปวางที่โฟลเดอร์  StartUp นี้ 
5. จบการติดตั้ง
